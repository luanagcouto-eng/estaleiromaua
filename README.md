# Estaleiro Mauá – Projetos Operacionais

Projeto contendo soluções e análises operacionais desenvolvidas para o Estaleiro Mauá, com foco em monitoramento, controle e melhoria de processos industriais.

## Objetivo

Este repositório reúne scripts, análises e documentações para:

- Monitoramento de processos produtivos no estaleiro mauá
- Controle e rastreabilidade de soldagem de juntas em módulos de plataformas  
- Consolidação e análise de dados operacionais  
- Criação de painéis para suporte à tomada de decisão  
- Automação de rotinas de análise de produção  

## Projetos Principais

- `Painel_Integrado_Maua.html`: Painel de controle para acompanhamento da soldagem das juntas dos módulos de plataformas  

## Funcionalidades

- Integração de dados de produção  
- Visualização de indicadores de soldagem  
- Acompanhamento de avanço físico de módulos  
- Identificação de gargalos operacionais  
- Suporte à gestão de produção
